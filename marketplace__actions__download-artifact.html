Action ID: marketplace/actions/download-artifact

Author: GitHub

Publisher: actions

Repository: github.com/actions/download-artifact

Description

Download a build artifact that was previously uploaded in the workflow by the upload-artifact action

Categories

Dependency management Build Packaging

Categorization

Primary: Dependency management

Confidence: high

Reasoning: This action is primarily used to manage dependencies by downloading build artifacts that are necessary for further steps in a workflow. It also relates to the build process as it deals with artifacts generated during builds.

Tags: artifact download build dependency workflow

Inputs

NameRequiredDescription
name No Name of the artifact to download. If unspecified, all artifacts for the run are downloaded.
artifact-ids No IDs of the artifacts to download, comma-separated. Either inputs `artifact-ids` or `name` can be used, but not both.
path No Destination path. Supports basic tilde expansion. Defaults to $GITHUB_WORKSPACE
pattern No A glob pattern matching the artifacts that should be downloaded. Ignored if name is specified.
merge-multiple No When multiple artifacts are matched, this changes the behavior of the destination directories. If true, the downloaded artifacts will be in the same directory specified by path. If false, the downloaded artifacts will be extracted into individual named directories within the specified path.
github-token No The GitHub token used to authenticate with the GitHub API. This is required when downloading artifacts from a different repository or from a different workflow run. If this is not specified, the action will attempt to download artifacts from the current repository and the current workflow run.
repository No The repository owner and the repository name joined together by "/". If github-token is specified, this is the repository that artifacts will be downloaded from.
run-id No The id of the workflow run where the desired download artifact was uploaded from. If github-token is specified, this is the run that artifacts will be downloaded from.

NameDescription
download-path Path of artifact download
