Description

Logs in the local Docker client to one or more Amazon ECR Private registries or an Amazon ECR Public registry

Authentication Infrastructure Packaging

Inputs

NameRequiredDescription
http-proxy No Proxy to use for the AWS SDK agent.
mask-password No Mask the docker password to prevent it being printed to action logs if debug logging is enabled. NOTE: This will prevent the Docker password output from being shared between separate jobs. Options: ['true', 'false']
registries No A comma-delimited list of AWS account IDs that are associated with the ECR Private registries. If you do not specify a registry, the default ECR Private registry is assumed. If 'public' is given as input to 'registry-type', this input is ignored.
registry-type No Which ECR registry type to log into. Options: [private, public]
skip-logout No Whether to skip explicit logout of the registries during post-job cleanup. Exists for backward compatibility on self-hosted runners. Not recommended. Options: ['true', 'false']
