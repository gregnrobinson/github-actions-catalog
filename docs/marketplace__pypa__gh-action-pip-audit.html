Description

Use pip-audit to scan Python dependencies for known vulnerabilities

Categories

Security Dependency management Code quality

Categorization

Primary: Security

Confidence: high

Reasoning: This action scans Python dependencies for known vulnerabilities, which directly relates to security. It also involves dependency management and contributes to code quality by ensuring that dependencies are safe.

Tags: pip-audit security dependency-check python vulnerabilities

Inputs

NameRequiredDescription
summary No render a Markdown summary of the audit (default true)

extra-index-urls No extra PEP 503-compatible indexes to use, whitespace separated
ignore-vulns No vulnerabilities to explicitly exclude, if present (whitespace separated)
disable-pip No disable pip
locked No audit lock files from the local Python project
internal-be-careful-allow-failure No don't fail the job if the audit fails (default false)
internal-be-careful-extra-flags No extra flags to be passed in to pip-audit
