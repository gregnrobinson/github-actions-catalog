Gitea Sync Secrets

Action ID: marketplace/appleboy/sync-secrets-action

Author: Bo-Yi Wu

Publisher: appleboy

Repository: github.com/appleboy/sync-secrets-action

Description

Copies secrets from the action's environment to many other repos.

Categories

Security Dependency management

Categorization

Primary: Security

Confidence: high

Reasoning: This action is focused on copying secrets, which is a security-related task, and it also involves managing dependencies across multiple repositories by syncing secrets to them.
