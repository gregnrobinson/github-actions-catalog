update_packager_index No By default, this action will update packager's indexes to avoid installation issues (`apt`/`brew` on linux/macOS respectively). install_ccache No By default, this action will install ccache with package manager. You can omit installation if you've already installed ccache and it is accessible in the shell

windows_compile_environment No Specify which compiler environment you are going to use on Windows image. Note that this field (a single value) is mandatory if you use this action on a Windows image. Available options are: ['msvc', 'msys2']
api_token No Token for using GitHub API
restore_cache No Whether to restore the cache at the start of this action
store_cache No Whether to store the cache at the end of job execution
remove_stale_cache No Whether to remove previous/stale cache entries after store cache completed. This requires parameter `api_token` to be valid. This requires `actions: write` permission. override_cache_key No Override cache key which is used for storing/retrieving the cache. Accept a string. Leave it empty to use the default value
