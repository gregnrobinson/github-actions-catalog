Discord Message Notify

Action ID: marketplace/appleboy/discord-action

Author: Bo-Yi Wu

Publisher: appleboy

Repository: github.com/appleboy/discord-action

Description

Send a message to a Discord channel using a webhook. This action allows you to notify a Discord channel by sending a message through a webhook.

Categories

Communication Notification

Categorization

Primary: Communication

Confidence: high

Reasoning: This action is designed to send messages to a Discord channel, which clearly falls under communication and notification categories.

Tags: Discord notification webhook message communication

Inputs

NameRequiredDescription
webhook_url No The URL of the Discord webhook. This is the endpoint where the message will be sent.
webhook_id No The ID of the Discord webhook. This is a unique identifier for the webhook.
webhook_token No The token of the Discord webhook. This token is used to authenticate the webhook.
message No The content of the message to send. The message can be up to 2000 characters long.
file No The content of the file to send. You can attach a file to the message.
color No The color code for the embed message. This sets the color of the embed's side border.
wait No Wait for server confirmation before responding. If set to true, the action will return the created message body.
tts No Set to true if the message should be read aloud using text-to-speech. This enables the text-to-speech feature for the message.
username No Override the default username of the webhook. This allows you to set a custom username for the message sender.
avatar_url No Override the default avatar of the webhook. This allows you to set a custom avatar for the message sender.
debug No Enable debug mode. If set to true, the action will output the request and response details.
