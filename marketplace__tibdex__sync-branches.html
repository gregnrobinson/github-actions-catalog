Sync branches

Action ID: marketplace/tibdex/sync-branches

Author: Thibault Derousseaux <tibdex@gmail.com>

Publisher: tibdex

Repository: github.com/tibdex/sync-branches

Description

Automatically merge a repository branch into other branches to keep them in sync.

Categories

Dependency management Communication Code quality

Categorization

Primary: Dependency management

Confidence: high

Reasoning: The action is designed to keep branches in sync, which falls under dependency management, as it ensures that different branches maintain consistent code. Additionally, it involves creating pull requests, which relates to communication and code quality.

Tags: sync branches GitHub automation pull requests
